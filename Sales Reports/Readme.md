# Sales Reports

This folder contains sales-related analytical reports built using Excel and Power Pivot.

## Reports Included

• Customer Market Performance Report  
• Division Performance Report  
• Market Performance vs Target Report  
• New Product Performance Report  
• Top 5 Countries Sales Report  
• Top 10 Products Report  
• Top and Bottom Products Report  

## Purpose

These reports help analyze:

• Customer-level performance  
• Market trends  
• Product performance  
• Sales growth patterns
